# SANS APPAREL - Enterprise Attendance Core System

A premium, production-optimized responsive real-time attendance logging system engineered specifically using vanilla web technologies (**HTML5**, **CSS3 Custom Properties**, and modern modular **ES6 JavaScript**), backed via **Supabase PostgreSQL & Auth Instance Layer Nodes**.

## Architecture & Integration Specs

- **Zero Bundling Overhead:** Runs raw natively inside modern web clients via modular ES6 execution engines (`type="module"`).
- **Enterprise Design Language:** Built upon SANS APPAREL branding visual primitives rules utilizing a clean, corporate `#16A34A` palette grid with soft spatial elevations shadows overlays.
- **Bi-Directional Security:** Real-time role-based tracking pipelines enforcement matching data validation matrices on Supabase level constraints rules.

## Onboarding / Deployment Routing Directives

1. Execute the configuration schema logic located inside `sql/database.sql` directly into your Supabase project query runner console block inside dashboard infrastructure views panel instances.
2. Open `js/supabase.js` and modify your custom endpoints mapping parameters credentials configurations strings (`SUPABASE_URL` and `SUPABASE_ANON_KEY`).
3. Deploy directly onto **Vercel** platforms or alternative web hosting systems without requiring compilation parameters builders triggers.
